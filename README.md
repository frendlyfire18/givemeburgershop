## Как запустить

Откройте в своем любимом редакторе папку с данным приложением и запустите команду:
```bash
npm init
```
после того как все зависимости установятся, запустите приложение с помощью вашего любимого менеджера пакетов:

```bash
yarn dev
# или
yarn start
# или
npm dev
# или
npm start
```
